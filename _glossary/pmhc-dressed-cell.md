A pMHC-dressed cell is a cell that has acquired peptide-loaded MHC molecules from another cell via trogocytosis, and can subsequently present that antigen to other cells as if it were an antigen-presenting cell.

This is best documented in TCR-affinity- and antigen-specific trogocytosis by CD8 T cells, which generates pMHC-dressed CD8 T cells that are then targeted for fratricide by neighboring CD8 T cells of the same specificity.
